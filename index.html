Towntest
for Minetest

Grow a town in minetest.

Download GitHub Minetest CoRNeRNoTe

Features

  • chest that allows you to choose a building, then you fill the chest with required items and an NPC will build the building more buildings can be added to the buildings folder and will be automatically added in the game
  • coming one-day - more NPCs - miners, lumberjacks, crafters

Docs

Requirements

  • Minetest >=0.4.3

Installation

Download and uncompress the latest version, and move the the minetest-towntest-master folder into your minetest/mods folder.

Usage

You will find the chests scattered in your world. Once you find one, choose a construction and fill up the materials required.
